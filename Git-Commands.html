Using GitHub

To add a file in your GitHub Account you have to follow the following steps:

  • Instaling Git
    • First you need to install git from the following url:
    • Git-Desktop
  • Creating Repository
    • In your GitHub account create a new repsitory
    • in its code section you can will see git url copy it
  • Creating File
    • Now create an html file in VS Code
  • Git Commands
  • You have to enter the following commands in terminal of VS Code:
    • git init *will initialize git*
    • git add "file_name" *will stage the file*
    • git remote add origin "url_of_your_repository" *will link your current folder with you online repository*
    • git commit -m "any_message_along_with_commit" *will commit your file in repsitory*
    • git push -u origin*will push your file to the online repository*
  • If you want to create multiple branches enter following commands:
    • git branch new_branch_name *creates a new branch*
    • git checkout new_branh_name *move you to the new branch*
    • git checkout master *moves you back to the master branch*
    • git merge new_branch_name *this will merge your branch work to the master branch*
