Using GitHub

To add a file in your GitHub Account you have to follow the following steps

  • Instaling Git
    • First you need to install git from the following url:
    • Git-Desktop
  • Creating Repository
    • In your GitHub account create a new repsitory
    • in its code section you can will see git url copy it
  • Creating File
    • Now create an html file in VS Code
  • Git Commands
  • You have to enter the following commands in terminal of VS Code
    • git init *will initialize git*
    • git add "file_name" *will stage the file*
    • git clone "url_of_your_repository" *will clone your current folder with you online repo*
    • git commit -m "any_message_along_with_commit" *will commit your file in repsitory*
    • git push *will push your file to the online repository*
